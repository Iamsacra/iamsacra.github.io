Scripts Script automount Editamos o ficheiro que executa os scripts at login, esse ficheiro é o rc.local em etc: vim /etc/rc.local Dentro do ficheiro fazemos o seguinte o seguinte: #! /bin/sh -e # espera 10 segundos sleep 10 # o/s script/s que queremos executar mount -t cifs -o username=user1,password=123 //192.168.107.52/Partilha /mnt/PC_Windows/ [o script que queremos] Agora temos que atribuir a premissão de execução com: chmod +x /etc/rc.local e agora quando reiniciarmos o debian vai executar o script O que podemos fazer é criar o mesmo script dentro de um ficheiro sh, por exemplo script.sh, e colocar dentro do rc.local. para podermos ter mais que um script a correr no pc. #!/bin/sh -e /root/scripts/autoMount.sh /root/scripts/nat.sh # E outros exemplos de scripts Não podemos esquecer de também colocar este script com as permissões de execução. NAT Script O script do NAT para o iptables é igual ao comando, mas está dentro do script para ser executado sempre que iniciarmos o computador, portanto é assim: #!/bin/sh iptables --table nat --append POSTROUTING --out-interface enp0s3 -j MASQUERADE
