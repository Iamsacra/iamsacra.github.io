Sem compressão Para agrupar ficheiros ou pastas sem comprimir, na pasta que estamos posicionados utilizamos o seguinte comando: tar -cvf arquivo.tar ficheiros_e_pastas -c: significa create -v: significa verbose, este não é obrigatório usar, mas é útil uma vez que nos diz todos os ficheiros que foram adicionados ao arquivo. -f: significa file arquivo.tar: nome que queremos dar ao arquivo tar ficheiros_e_pastas: por fim adicionamos todas os ficheiros e pastas que queremos para colocar no arquivo tar. Se quisermos indicar outro caminho onde criar o arquivo, escolhemos esse caminho, por exemplo: tar -cvf /home/linux/arquivo.tar ficheiros_e_pastas Para extrair um ficheiro tar, utilizamos o seguinte comando: tar -xvf arquivo.tar Para colocar um ficheiro dentro de um ficheiro ou pasta já agrupado, podemos usar o seguinte comando: tar -rvf arquivo.tar outros_ficheiros_e_outras_pastas -r: significa append, ou seja acrescentar ao ficheiro. Para listar os ficheiros dentro do arquivo usamos o seguinte comando: tar -tvf arquivo.tar Com compressão Os comandos neste caso são semelhantes, só tendo que acrescentar um z. tar -zcvf arquivo.tar.gz ficheiros_e_pastas -z: significa gzip, outro meio de compressão que iremos falar mais para a frente. .tar.gz: deixamos de usar apenas tar e acrescentamos uma vez que é a extensão padrão do gzip. Para extrair para a pasta atual usamos o comando: tar -zxvf arquivo.tar.gz Para extrair para outra pasta: tar -zxvf arquivo.tar.gz:C /root/ -C: significa directory, usado para apontar para outro diretório. Para listar o conteúdo de um arquivo, o comando é igual se for comprimido ou não: tar -tvf arquivo.tar.gz
