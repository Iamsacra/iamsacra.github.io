ZIP A sintaxe do comando é a seguinte: zip arquivo.zip arquivos_ou_pastas_a_comprimir Para descompactar usamos o comando unzip: unzip arquivo.zip O comando unzip não cria um pasta por defeito, para colocar todo o conteúdo do zip temos que criar a pasta previamente. Para listar o conteúdo de um ficheiro “zipado” usamos a opção –sf, tal como: zip -sf arquivo.zip O comando zip só comprime a pasta e não todo o seu conteúdo dentro dele, para comprimir a pasta e todo o seu conteúdo temos que usar a opção –r: zip -r arquivo.zip /home/
