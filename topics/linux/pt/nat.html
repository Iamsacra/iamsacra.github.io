NAT Precisamos de uma placa configurada para distribuir o NAT, e outra com acesso a internet, um servidor e Linux - DHCP configurado. Temos que transformar o linux num rooter e transferir o trafego da placa interna para a com acesso a internet. temos que descometar a seguinte linha no ficheiro /etc/sysctl.conf (linha 28) net.ipv4.ip_forward=1 Depois disso temos que reinciar o servidor para assumir as alterações De seguida instalamos o iptables com o seguinte comando: apt install iptables Depois disso devemos de usar o seguinte comando: iptables --table nat --append POSTROUTING --out-interface enp0s3 -j MASQUERADE Porem sempre que reiniciarmos o servidor tinhamos que refazer este comando, mas podemos fazer um script que corre sempre que iniciamos o servidor.
